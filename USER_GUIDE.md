## Troubleshooting

### Theme Not Saving
- Make sure cookies/localStorage are enabled in your browser
- Try logging out and back in
- Clear your browser cache if issue persists

### Language Not Changing
- Refresh the page
- Check that JavaScript is enabled
- Try a different browser if problem continues

### Activity Logs Not Showing
- Make sure you're logged in
- Check your internet connection
- Try refreshing the page
- Contact support if issue persists

### Mobile Menu Issues
- Make sure you tap directly on buttons
- Try landscape mode if items are hard to tap
- Refresh the page if menu doesn't close
